# Support CRM System

A full-stack Support CRM application built using FastAPI and React.

## Features

- Create support tickets
- Update ticket status
- Search and filter tickets
- Add notes to tickets
- Responsive dashboard UI
- REST API with FastAPI
- React frontend with Tailwind CSS

---

## Tech Stack

### Backend
- FastAPI
- SQLAlchemy
- SQLite

### Frontend
- React
- Vite
- Tailwind CSS
- Axios

## Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
